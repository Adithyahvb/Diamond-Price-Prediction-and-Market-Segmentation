# Diamond-Price-Prediction-and-Market-Segmentation
With given features building regression models to predict the price of the diamond and using k means clustering market segmenting the diamond

FEATURES GIVEN:

Carat (Weight)

Cut (Quality of Shape & Light Performance)
  
  Categories (best → worst):
  •	Ideal
  •	Premium
  •	Very Good
  •	Good
  •	Fair

Color (Whiteness of Diamond)
  
  Scale:
  
  •	D → Colorless (best)
  •	E–F → Near colorless
  •	G–J → Slight color

Clarity (Internal Purity)
   
   Scale (best → worst):
   
  •	IF (Internally Flawless)
  •	VVS1, VVS2 (Very very slight inclusions)
  •	VS1, VS2
  •	SI1, SI2
  •	I1 (Included)

Depth - Height of the diamond relative to its width
  
  •	Unit: Percentage (%)

Table - Width of the top surface of the diamond
  •	Unit: Percentage (%)
  
Dimensions (x, y, z)

Price
